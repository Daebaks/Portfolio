React App Movies + Blogs

multi-theme website to create blogs and rate movies

It is a simple user interface where users can search for favorite movies and submit ratings, and users can create blogs. Blogs and movies ratings are saved/fetched from a Firebase data base that is synced with the project. The website is mobile friendly and has light and dark themes.

Movie search funtionality is connected to the omdbapi.com API, and the homepage has two components to display saved blogs and ratings from th edatabase storage.

Each blog/rating details can be viewed in a separate page. User also can delete any of the created/saved blogs or ratings.
